# Delivery Route Planner

Delivery Route Planner is a web application designed to help organize daily delivery routes for field workers.
The goal of the app is to quickly create a list of delivery locations and send the final route through WhatsApp.

The application is optimized for mobile use and focuses on simple interactions such as tapping and drag-and-drop instead of typing.

## Features

* Select laboratories and clinics from predefined lists
* Build a delivery route using drag and drop
* Reorder route items easily
* Add optional icons or delivery time to specific stops
* Send the route to workers via WhatsApp
* Manage routes for multiple workers
